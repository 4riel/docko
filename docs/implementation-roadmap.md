# Implementation Roadmap

## Phase 1: Publishable Core

- ship schemas for registry and session manifests
- implement core claim, release, heartbeat, and stale recovery logic
- ship CLI with stable exit codes
- ship generated mirror rendering
- publish README and protocol docs

## Phase 2: Claude Code First-Class Support

- hook adapters
- command wrappers
- settings examples
- Agent Teams delegation docs
- workspace guidance skill docs

## Phase 3: Additional Runtime Adapters

- Codex helper scripts and session guidance
- Aider wrapper examples
- Cursor CLI / Cursor Agent mapping
- OpenCode and generic shell examples

## Phase 4: Hardening

- conformance fixtures
- corrupted-registry recovery tooling
- optional `doctor` and `repair` flows
- macOS performance benchmarks
